# Cài đặt Golang

Để bắt đầu lập trình Go, bạn cần tải về công cụ biên dịch (Go Compiler) và thiết lập môi trường.

## 1. Tải về

Truy cập trang chủ [https://go.dev/dl/](https://go.dev/dl/) và chọn bộ cài đặt phù hợp với hệ điều hành của bạn (Windows, macOS, hoặc Linux).

## 2. Kiểm tra cài đặt

Sau khi cài đặt xong, mở terminal/command prompt và chạy lệnh sau:

```bash
go version
```

Nếu thành công, bạn sẽ thấy phiên bản Go hiện tại (ví dụ: `go version go1.23 windows/amd64`).

## 3. Cài đặt VS Code

Mặc dù bạn có thể dùng bất kỳ text editor nào, **Visual Studio Code (VS Code)** kết hợp với extension **Go** của đội ngũ Google là lựa chọn phổ biến nhất.
